- 所有插件均遵循 Vite Plugin 规范，通过 `name` 字段标识身份，并在 `configureServer` 中安全地扩展 `server.middlewares` 或 `server.httpServer`。
- 路径处理统一使用 `node:path` 进行标准化，并对用户输入的路径片段执行严格的安全校验（如禁止 `..`、限制在 `projectRoot` 内），防止目录穿越攻击。
- HTTP 响应封装采用统一的 `sendJson` 辅助函数，自动设置 `Content-Type` 为 `application/json` 并禁用缓存，确保 API 行为一致性。
- 对于需要懒加载的重型依赖（如 `extract-zip`），采用模块级变量缓存结合 `import()` 动态导入的方式，避免在 Vite 配置阶段产生不必要的性能开销。
